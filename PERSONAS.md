# User Personas

_Last updated: 2026-06-13_

## Sarah — Mid-Market E-Commerce Merchant

**Role:** E-commerce store owner / Director of Digital
**Goals:** Improve checkout conversion rates, understand why customers abandon carts, reduce fraud losses
**Pain Points:** Existing checkout platforms share her customer data; limited insight into what's causing cart abandonment; high interchange fees
**Technical Level:** Medium — comfortable with dashboards, not a developer
**Uses:** Merchant dashboard, analytics widgets, AI insights, configuration wizard
**Plan Tier:** Pro ($299/mo)

## Marcus — E-Commerce Technical Lead

**Role:** Lead Engineer at mid-market retailer
**Goals:** Migrate from legacy checkout without downtime; integrate intellipay API; validate ML improvements
**Pain Points:** Migration risk; integration complexity; verifying AI actually improves conversion
**Technical Level:** High — uses API directly, reviews analytics via custom queries
**Uses:** REST API, Shadow Mode migration tooling, webhook integrations, custom analytics queries
**Plan Tier:** Enterprise

## Emma — SMB Merchant

**Role:** Solo e-commerce founder ($200K/year store)
**Goals:** Reduce checkout friction, accept NFC payments, keep costs low
**Pain Points:** Can't afford enterprise SaaS; privacy concerns about data sharing; too busy to manage complex tools
**Technical Level:** Low — needs plug-and-play widget
**Uses:** Checkout widget (JS snippet), basic dashboard, TaptoPhone NFC
**Plan Tier:** Starter ($49/mo)
